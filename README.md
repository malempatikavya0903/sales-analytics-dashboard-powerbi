# 📊 Sales Analytics Dashboard

## Overview

This project is an interactive Sales Analytics Dashboard built using **Power BI** to analyze sales performance, customer behavior, and business profitability. It enables users to monitor key business metrics and gain actionable insights through interactive visualizations.

---

## Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## Key KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Quantity Sold

---

## Dashboard Features

- Interactive slicers for Category, Subcategory, Region, State, City, and Year
- Monthly and Yearly Sales Trend Analysis
- Customer Performance Analysis
- Product Performance Analysis
- Regional Sales Analysis
- KPI Cards for quick business insights

---

## Business Insights

- Identified top-performing products based on sales and profit.
- Analyzed customer purchasing trends.
- Compared regional sales performance.
- Tracked business growth over time using interactive visuals.
